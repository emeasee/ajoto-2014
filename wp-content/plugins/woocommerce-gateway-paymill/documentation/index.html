Install

 

To install the plugin, go to your WordPress Administration, and follow these steps:

  1. Click on Plugins
  2. Click on Add New
  3. Below the headline "Install Plugins" click Upload
  4. Click browse and find the zip file you downloaded from codecanyon
  5. Click Install Now
  6. Click acvtivate now when it is finish uploading

The plugin is now installed and activated

Now go to WooCommerce > Settings and click on the Payment Gateways tab. In here you the possibility to activate the plugin and enter alle the needed API settings.
