The SeedProd WooCommerce MailChimp Subscribe WordPress plugin displays a newsletter signup checkbox on checkout for WooCommerce integrated with MailChimp. Also includes a MailChimp List Health Dashboard Widget.

Installation

There are several methods to install a WordPress plugin. Choose a method below.

New Install via FTP

  1. Unzip the package woocommerce-mailchimp-subscribe-x.x.zip on you computer.
  2. Upload the unzipped folder the /wp-content/plugins/ directory.
  3. Activate the plugin through the Plugins menu in WordPress.
  4. Goto WooCommerce > Email Seetings to set your options.

New Install via the WordPress uploader

  1. Click Plugins > Add New inside of your WordPress install.
  2. Click Upload and select the package woocommerce-mailchimp-subscribe-x.x.zip
  3. Activate the plugin through the Plugins menu in WordPress.
  4. Goto WooCommerce > Email Seetings to set your options.

Updating via the FTP

  1. If you've made any custom changes to the core you'll need to merge those changes into the new package.
  2. Unzip the package woocommerce-mailchimp-subscribe-x.x.zip on you computer.
  3. Replace the current folder in your plugins folder with the new unzipped folder from your computer.

Updating via the WordPress uploader

  1. Navigate to the Plugins page in WordPress and delete the current plugin.
  2. Now follow the New Install via the WordPress uploader instructions.
